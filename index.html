reqres.in API Testing Project

This repository contains a Postman project for testing the reqres.in API. The project includes a collection with requests to various API endpoints, along with test scripts to validate the responses.

Getting Started

  • Clone this repository.
  • Import the Postman collection (reqres_in_api_testing.postman_collection.json) into Postman.
  • Run the requests in the collection to test the API.

Collection Details

Collection Name: User Management

API Schema: OpenAPI (Using https://reqres.in/docs/swagger.json)
